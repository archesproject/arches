The Report Designer allows you to fine-tune many aspects of this Report. By default it is populated with all of the Cards in your Resource Model, each of which becomes a section in the Report. To exclude a section from publication, you can select it in the contents panel at left and toggle the Publish Section control. You can also change the order of the sections by dragging them in the contents panel, and relabel any section or its internal parts.

Configuring Headers

To set a Report as the active one for your Resource Model, you'll need to select the Header section in the contents panel, and use the "Active" switch. Only one Report may be active at a time, so any other Reports associated with this Resource Model will be set to "inactive". Other Header settings are described below.

For All Templates

Report Title: add a title which will appear at the top of the report

For the Map Header Template

Map Controls: show hide the map controls, such as basemap switcher or legend (note that you can use these map controls to set the default basemap before you disable them)
Position: set the default map center
Zoom: set the default zoom level
Geocoder: choose which geocoding engine to use, or disable if desired
Resource Properties: set the color of the drawing tools, and choose which geometry types are allowed

For the Image Header Template

Included Image Nodes: select the nodes in this Resource Model in which images will be found to display in the header
