The Menu Designer helps you configure the data entry Menus for your Resource Model. You will need to define some settings for the Menu, and then add some Cards to it. When you have saved your edits, click "Back to Menus" to return to the Menu Manager, or find a new Menu to edit in the dropdown.

Menu Settings

In the right-hand panel, you will define some basic settings for the Menu.

Identifiers

Name: add a name to this menu
Subtitle: add a subtitle to this menu

Icon

Icon: select an icon for this menu

Status

Default Visibility: set whether or not this menu should be visible by default

Adding Cards to the Menu

To add a Card to this Menu, you must first find it in the Data Card Library. This is a listing of all the Cards associated with your Resource Model, so if you don't see a Card that you are wanting to use, you'll need to head to the Graph Manager or Card Manager to add that Card's Branch. When you've found the Card you want, just drag it into the center panel.
